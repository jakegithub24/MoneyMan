# Input validation

## Onboarding input validation
1. Cloud Sync
    - Cloud Username : `a-z, 0-9`, max 30 chars
    - Cloud Password : alpha-numeric + special characters, max 30 chars
2. New Profile
    - Full Name : `A-Z, a-z`, 'First' + ' ' + 'Last', max 30 chars
    - Username : `a-z, 0-9`, max 30 chars
    - Profile Picture : 256x256px, <=5mb
    - Cloud Password : alpha-numeric + special characters, max 30 chars
